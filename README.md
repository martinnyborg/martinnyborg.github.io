# jenkins-theme

> Cloudinsight 使用的简单的 Jenkins 自定义主题

![截图](./screenshot.png)

## 修改内容

- 替换 Jenkins 的 Logo
- 修改默认字体

## 安装

- Jenkins 需要安装这个插件

```
http://wiki.jenkins-ci.org/display/JENKINS/Simple+Theme+Plugin
```

- 设置自定义的 CSS 地址

![配置界面](./plugin_settings.png)

```
http://d.oneapm.com/static/jenkins-theme/2/style.css
```

如果需要可以加入 GA 统计的代码

```
https://c.oneapm.com/gtm.js
```
